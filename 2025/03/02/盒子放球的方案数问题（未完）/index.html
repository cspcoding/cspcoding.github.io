盒子放球的方案数问题

发表于2025-03-02|更新于2025-03-06|组合数学
|阅读量:

本帖学习要求的预备知识是组合数、第二类Stirling数以及递归和递推（会做1206题）等。考CSP初赛只需要掌握所有的理论分析，会做试卷即可；考GESP8级或者考CSP复赛需要掌握到程序实现。

在做算法题的过程中，很多计数类问题的很多情况可以转化为如下模型：\(q\)个相同/不同的球放到\(h\)个相同/不同的盒子中，盒子允许/不允许为空，问有多少种放入方案。

8个模型过于常见，本文进行总结，以及附带了它们的拓展模型，如每个盒子至少t个球、每个盒子至多t个球、非空盒子不能相邻等等。

1）有如下文字需要先悉知：

  • "无标号" = "相同"，"有标号" = "不同"
  • “盒子相同”＝“可以围成一个圈，不分谁是第一第二”，“盒子不同”＝“盒子摆成一条线，要区分前后”

２）为避免同学们记晕，特意把球的数量定为ｑ（ｑｉｕ），盒子的数量定为ｈ（ｈｅ）。

３）下面的八种基本题型是按照难易程度和认知的先后为序来进行介绍的。若有部分模型是建立在其他模型理解的基础上的，则按照认知的顺序来排序。

一、八种基本题型

1.1 模型１：球不同，盒子不同，允许空（\(h^q\)）

1.1.1 分析

　　这个题和其他７个题型的解题思路都不同，且很易懂，就摆在第一个了。

　　每个球都有编号，\(h\)个盒子排成一排，允许为空。既然每个球都不同，那我们可以把\(q\)个球的放置过程分为\(q\)个步骤，在每个步骤中，每个球都有\(h\)个选择，不受上一个球方案的影响（因为允许盒子为空）。根据乘法原理，整个方案数为\(h*h*...=h^q\)。

1.1.2 练习

1.1.2.1 理论题

　　８个不同的球放入标号为１、２、３的３个盒子中，问有多少种不同的摆法？

　　解：根据分析，有\(3^8=9^4=81^2=6561\)种摆法。

1.2 q球同、h盒不同、不允许空（\(C_{q-1}^{h-1}\)）

1.2.1 分析

　　问题模型：把\(q\)个无标号的球放到\(h\)个有标号/有序的盒子中，盒子不允许为空，求方案数。这种题型还有另外一个问法：“求\(x_1+x_2+...+x_h=q\)这个方程的正整数解的数量”。

　　这个题型，如果\(q<h\)，放不下，直接输出为0，表示无解。

　　接下来我们考虑把\(q\)个球分进\(h\)个盒子里。因为盒子是不同的，也就是说：假如８个相同的球分成３堆，则1,3,4和4,1,3是不同的且都合理的方案，有顺序之分。这种情况下，如果采用“数的分解”的思路的话，就需要更为复杂一些的深度优先搜索来进行穷举，代码会比较复杂。其实这里还有一种更为简单的做法，那就是采用隔板法：我们把\(q\)个球摆成一条线，那么\(q\)个球之间有\(q-1\)个空隙，因为用\(x\)个无差别的板子可以把摆成一条线的多个球分为\(x+1\)堆，所以现在要分为\(m\)份则需要\(m-1\)个无差别的板子来隔开，如下图所示。

　　隔开后从左往右依次每一份内的球的个数就对应着一个盒子内装的球数。这种分法自然允许出现4,1,3的情况。故原问题可以等价为“在\(\textcolor[RGB]{255,0,0}{q-1}\)个空位选出\(\textcolor[RGB]{255,0,0}{h-1}\)个空位来装插板”，也就是\(\textcolor[RGB]{255,0,0}{q-1}\)个中选\(\textcolor[RGB]{255,0,0}{h-1}\)个（空位），即\(\textcolor[RGB]{255,0,0}{C_{q-1}^{h-1}}\)。剩下的事情就是去求组合数，问题就很简单了。

1.2.2 练习

1.2.2 1 理论题

　　8个相同的球放入标号为1、2、3的三个盒子中，每个盒子中至少有一个。问有多少种不同的放法？（隔板法）  解析：8个球排成一排有7个空隙，在7个空隙中任取两个来分别插入１个隔板，有\(C_7^2=7*6/(1*2)=21\)种。这样将8个球分成三堆，第一堆放到1号盒子内，第二堆放到2号盒子内，第三堆放到3号盒子内，即可解决。

　　结论：\(q\)个相同的球放入\(h\)个不同的盒子中\((q\geq h)\)，且不能有空盒的放法数是\(C_{q-1}^{h-1}\)。

1.3 q球同、h盒不同、允许空（\(C_{q+h-1}^{h-1}\)）

1.3.1 分析

　　我们先在上一节讲了模型３，现在倒回来讲模型２，因为模型２是模型３的拓展。模型２允许盒子为空，我们可以想办法把它变成一个不允许为空的题型。方法很简单，那就是：我们可以先从其他地方借来\(h\)个相同的球，和原来的\(q\)个球一共构成\(q+h\)个无差别的球，然后再采用插板法，把它们分成不允许为空的\(h\)堆（利用1.2节模型的知识），然后再把原来借来的那\(q\)个球拿走。这样之后，如果有些堆里的球都是借来的，那还回去之后，该盒子里就成了空盒，和题意吻合。

​ 按照上面的分析，在\(q+h\)个无差别的球之间使用插板法，即在 \(q+h-1\)个空位中插入\(h-1\)个板子，从而把球分成\(h\)堆，一共有\(C_{q+h-1}^{h-1}\)种选择法，这就是本题的答案。

##　1.4 球不同，盒子相同，不允许空\((\ Stirling_2(q,h)\ )\)

1.4 1 分析

　　球不同，意味着每个球都有编号；盒子相同，意味着盒子是无序的，你可以把盒子们围成一圈，甚至胡乱摆放。这是个什么模型？仔细思考，可以发现，这个模型很类似于“把\(q\)个不同的数划分成\(h\)个无序的集合”的问题。其基本思路方向是：“对于每个数，有两大类选择：１是自己成为一个新集合，２是选择一个现有的集合并把自己加入其中”。这个思路是一个和递推递归有关的东东，涉及到了dp。

　　把dp的那几板斧搬过来：

　　１）状态定义：dp[i][j]表示把前i个不同的球（数）放入前j个相同的盒子（集合）的不同的放法方案数；

　　２）所求目标：dp[q][h]

　　３）状态转移方程：

　　　　分析：对于第\(i\)个球（数），有两种选择:

​ 　　a. 若先把前\(i-1\)个球放到前 \(j-1\)个盒子里且无空盒（根据定义，有dp[i−1][j−1]种摆法），再把第 \(i\) 个球单独放在一个新的盒里（只有1种摆法），所以选择ａ的方案数根据乘法原理有\(dp(i, j) =1×dp(i−1,j−1)\) ；

​ 　　b. 若先把前\(i-1\)个球放到前 \(\textcolor[RGB]{255,0,0}j\)个盒子里且无空盒（根据定义，有dp[i−1][j]种摆法），再在这\(\ j\ \)个盒子中任选1个盒子，把第$ i$ 个球放进去（有\(C_j^1=j\ \)种摆法），所以选择ｂ的方案数根据乘法原理有\(dp(i, j) = dp(i−1,j) × j\)。

​ 　　　联合a）和b），根据加法原理，得到递推方程式如下： \[ dp(i, j)=dp(i−1,j−1)+dp(i−1,j)×j \] 　　　　这个方程有一个很著名的名字，那就是赫赫有名的\(Stirling2\)数（第二类斯特林数）。

　　４）特例

  • 若只有1个盒子，就把所有的球全都放进去，这也算是1种方法，即dp[i][1]=1，\((i \in[1,q])\)
  • 若\(q<h\)，那就无法放入且盒子不空，即dp[q][h]=0，\((q < h)\)

1.4.2 练习

1.4.2.1 理论题

　　解：这属于“集合划分”的类型题，需要用第二类斯特林数。答案是\(Stirling_2(8,3)\)。这里需要在草稿纸上画表格来填写，只需要填写前８行前３列即可。

##　1.5 球不同，盒子相同，允许空\((\ \Sigma_{i=1}^{i=h} Stirling_2(q,\ i)\ )\)

1.5.1 分析

　　盒子相同，意味着肯定不能用插板法之类的了，因为插板法只适用于盒子不同时的情况。球不同，意味着这也不是放苹果的类型，因为放苹果是假设苹果无差别的。进一步来看，球不同，意味着要对每个球逐一处理。怎么处理呢？面对无差别的盒子，也就是面对着无序的若干个集合（即集合内无序，集合之间也无序），只能选择加入或者新开。那这个题和1.4节的题型很像了。那个Stirling２数是适用于“不允许空盒”的情况。而本题型是“允许为空”的情况，看能不能利用前者来实现本题型的求解。这个分析通过排除法，基本确定了深度思考的方向。

　　我们已经知道“\(q\)球不同，\(h\)盒子相同，不允许空”的方案数是\(Stirling_2(q,h)\)。如果现在允许有空盒，那无非就是以下几种情况之一：

1）有\(0\)个空盒，\(h\)个非空盒：该方案数＝\(Stirling_2(q,h)\)；

2）有\(1\)个空盒，\(h-1\)个非空盒：该方案数＝\(Stirling_2(q,h-1)\)；

3）有\(2\)个空盒，\(h-2\)个非空盒：该方案数＝\(Stirling_2(q,h-2)\)；

......

h）有\(h-1\)个空盒，\(1\)个非空盒：该方案数＝\(Stirling_2(q,1)\)；

不可能有“\(h\)个空盒，\(0\)个非空盒”的情况，那样就没地方放\(q\)个球了，方案数为０。

根据加法原理，本题的方案数＝这\(h\)种情况对应的方案数之和\(=\ \Sigma_{i=1}^{i=h} Stirling_2(q,\ i)\)。现在要求这么多的第二类斯特林数，用dp的方式建表来累加，是最好的办法。

1.5.2 练习

1.5.2.1 理论题

　　８个不同的球放入３个相同的盒子里，允许有空盒，问有多少种不同的方法？

　　解：根据1.4.4.1节提供的Stirling2表格（考试时需要自己画出来），\(Stirling_2(8,1)=1,\ \ Stirling_2(8,2)=127,\ \ Stirling_2(8,3)=966\)，所以本题的正确答案是\(1+127+966=1094\)种。

##　1.6 q球不同，h盒子不同，不允许空\((\ h!* Stirling_2(q,\ h)\ )\)

1.6.1 分析

　　这个题型和之前我们学过的“球不同，盒子相同，不允许空”（对应时间复杂度为\(Stirling_2(q,h)\)​）的题型只有一字之差，即本题型要求“盒子不相同”，换句话说就是盒子是有编号，有顺序之分的，是一个排列而不只是个组合。\(\{\ \{1\},\ \ \{2,3\}\ \}\)​和\(\{\ \{2,3\},\ \ \{1\}\ \}\)​在原来的题型中认为是同一个方案，但是在本题型中被认为是２个不同的方案。而且，因为每个球都有不同的编号，所以一个方案中每个盒子内的内容（编号序列）肯定是不同的，所以将这\(h\)​个序列之间（不是内部，内部仍然是无序的，即\({2,3}\)​和\({3,2}\)​仍被视为相同）可以做一个全排列，就会产生\(h!\)​种排列法。根据乘法原理，最终的方案数为\(\ \ h!* Stirling_2(q,\ h)\)。

1.5.2 练习

1.5.2.1 理论题

　　８个不同的球放入标号为１、２、３的３个盒子里，每个盒子里至少要有１个球，问一共有多少种不同的放法？

　　解：根据分析，放法数为\(3!*Stirling_2(8,3)=6*966=5796\ \)种。

1.7 q球同、h盒同、允许空（放苹果，朴素版）

1.7.1 分析

　　问题模型：把\(q\)个无标号的球放到\(h\)个无标号的盒子中，盒子允许为空，求方案数。

　　这个题型，其实和放苹果（1206）是高度相似的，解题思路是一样的；它也和“整数的分拆”（把整数ｑ分拆成ｈ个自然数的方案数）的思路是一致的。假设dp[q][h]表示“把q个相同的球放入h个相同的盒子里且允许盒子为空”的方案数。那么根据q和h之间的数值关系，可以有下面两种情况：

　　１）当q>=h时，有无空盒或者有空盒这两种情况

　　　　a) 有空盒：必然至少有１个空盒。因为所有的盒子都是相同的，所以这１个空盒在哪里并不重要，所以这种情况下的方案数和“把这１个空盒拿开之后”的方案数是一样的，所以有dp[q][h]=dp[q][h-1]。至于有多个空盒的话，仍属于dp[q][h-1]的情况以内，属于“先有一个空盒”之后的后续情况，所以不在这里深入分析。

　　　　b)无空盒：既然所有的盒子都至少装入了１个球，可以用胶水把这个球粘贴在盒子底部，当作盒子的一部分，装数方案中每个盒子中的数量都减少了１，问题就变成了“把ｑ－ｈ个球放入ｈ个盒子里，允许为空”（每个盒子要用掉１个球来贴，一共会少ｈ个球；原方案若某个盒子内只有一个球，粘贴之后相当于“盒内无球”了，所以新问题应允许盒子为空）。其方案数仍然和原问题的方案数是相等的。例如“4,4,2”减为“3,3,1”，“2,1,1”减为“1,0,0”，但仍是２个方案。所以dp[q][h]=dp[q-h][h]

　　这两种情况，适用于加法原理，所以在１）情况下有dp[q][h] = dp[q][h-1] + dp[q-h][h]。

　　２）当q<h时，必然有空盒的存在，且至少有１个。把多余的这１个空盒拿开，并不会影响原方案，也不会影响总方案数。所以有dp[q][h] = dp[q][h-1]。

　　综合情况１）和情况２），有dp[q][h] = dp[q][h-1] + dp[q-h][h]。　　

　　注意，不要理解成了dp[q][h] = 2 * dp[q][h-1] + dp[q-h][h]了，这不是相加的关系。

　　另外，为了提高效率，当q<h时，dp[q][h]实际上是等于dp[q][q]的。多出来的盒子必然是为空的（空盒可能不止这些，但这h-q个盒子肯定是空的），直接可以无视。

1.8 q球同、h盒同、不允许空（放苹果，粘贴版）

1.8.1 分析

　　问题模型：把\(q\)个无标号的球放到\(h\)个无标号的盒子中，盒子不允许为空，求方案数。

　　这个题型和模型０的唯一区别就是“不允许空”。但是，如果你想把２个球放入１００个盒子里，还要求盒子不能为空，这就有点过分了，所以当\(q<h\)时问题是无解的。

　　当\(q \geq h\)时，可以先在每个盒子里放入一个无差别的球，这样就还剩\(q-h(是\geq0的)\)个球了。那现在问题就变成了求“把\(q-h\)个相同的球然如\(h\)个相同的盒子里，允许盒子为空”的方案数了。

1.8.2.1 理论题

　　8个相同的球放入3个相同的盒子中，每个盒子中至少有一个。问有多少种不同的放法？

二、总结归纳

　　上述的八种题型，非常容易混淆。这里按照难度和关联性做排序，如下表所示（“不”表示“不相同”或“不允许”）。同学们可以在理解基础上尽量背住。

从上表中可以看出：

１）当球不同时，意味着要针对每个编号的球去分析，大概率要朝着“集合划分”的思路去想策略，要ｄｐ填Stirling2数；

２）当盒子不同时，意味着盒子有线性的顺序之分，排成一排，朝着“插板法”的思路去想策略；

３）若不满足１）和２），则只剩“放苹果”了，朝这个方向去想，要ｄｐ。

注意事项：

　　１）求方案数，ans一定要定义成long long类型，以免爆掉。

　　２）另外，用dp填表时，注意边界和初值，必要时不妨用代码把填写后的dp表格完整打印出来看看，尤其是输入值较大时的结果值。

三、拓展模型

约定拓展模型标号，包含原本模型是哪个（用数字表示），以及是第几个拓展模型（用大写字母表示）

3.1 模型8A （q球同、h盒同、不允许空，每个盒子至少要摆t个）

3.1.1 分析

　　这个模型是模型８的拓展。

　　１）如果\(q<h*t\)，则无解；

　　２）如果\(q \geq h*t\)，仍然可以沿用模型８的思路，给无差别的每个盒子预先装入\(t\)个无差别的球，还剩\(q-h*t\)个无差别的球需要摆入\(h\)个无差别的盒子里，问题变成了“\(q-h*t\)个球同，\(h\)盒同，不允许空”，可用模型７（注意，不是模型８）来解决。

3.1.2 练习

１.　１０个无差别的球放入３个无差别的盒子里，要求每个盒子至少要有２个球，不同摆法的方案数是多少？

　　解：这是在不允许为空的基础上，要求每个盒子至少要有\(t=2\)个球，问题等同于“\(10-3*2=4\)个无差别球放入\(3\)个无差别盒子里，允许盒子为空”，也等同于“把４分成３个无序的整数相加，整数可以为０”，其方案为“\(\ \{0,0,4\},\ \ \{0,1,3\},\ \ \{0,2,2\},\ \ \{1,1,2\}\ \)”这４种。也可以写出转移方程，填表，取到dp[4][3]而得出。

２.　８０个无差别的球放入１５个无差别的盒子里，要求每个盒子至少要有４个球，不同摆法的方案数是多少？

　　写程序填表，取dp[20][15]得到615，即为正确答案。
