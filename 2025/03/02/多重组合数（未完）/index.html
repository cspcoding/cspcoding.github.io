1.1 [GESP202312 八级] 奖品分配

　　

1. 问题核心

题目要求计算将N个奖品分配给N个同学的方案数，其中奖品分为M种，每种的数量为\(a_i\)，同种奖品无差别，不同位置的同学有差别。总奖品数\(S\)满足\(N ≤ S ≤ N+1\)。需要分两种情况处理：

  • 当S=N ：所有奖品恰好分完，方案数为多重组合数：\(\frac{N!}{(a_0)! \ \ \cdot\ \ (a_1)!\ \ \cdot \ \ \ \ldots \ \ \ \cdot \ (a_{M-1})!}\)。
  • 当S=N+1 ：必须有一个奖品剩下，此时方案数为\(\frac{(N+1)!}{a_0! \cdot a_1! \cdot \ldots \cdot a_{M-1}!}\)。

2. 数学推导

  • S=N的情况 ：直接应用多重组合公式，计算阶乘的除法。
  • S=N+1的情况 ：通过数学推导发现，总方案数等于将总奖品数视为N+1的多重组合数。这是因为每个奖品ai的贡献可以合并为总和的阶乘除以各ai的阶乘。

3. 模运算优化

  • 预计算阶乘 ：预先计算0到MAXN的阶乘，取模\(10^9+7\)。
  • 逆阶乘计算 ：利用费马小定理，通过快速幂计算阶乘的逆元，存储在数组中。

4. 示例说明

  • 输入 ：N=2，M=2，a=[2,1]，总奖品数S=3（即N+1=3）。

  • 计算

    ：

    • 分子为3! = 6。
    • 分母为2! * 1! = 2。
    • 结果为6/2=3，与实际分配方式一致。
