1.P1177 【模板】排序

强调补充说明：

1） 当左指针和右指针重叠或者错位时，退出while循环，所以while循环条件应写为while ( i < j ) ，不要加入’=’号！

2）对x的取值，不一定非要去取区间中段的值，可以随机取（仍然无法确保肯定能把区间分成长度接近的两段）。

3）如果要取区间中段值，为防止int 或者long long的加法溢出，最好是写成 x = a[ le + (ri - le)/2 ]的代码形式。

4）当退出while循环时，i 和 j 正好是重叠的(i == j )，或者是错位的( j+1 == i )。此时往下递归，左区间取[le, j ]，右区间取[j+1, ri ]（不能取[i, ri]，因为i和j可能是重叠的，若取 i 会让 a[i] 同时在两个子区间内）

5）代码模板建议背下来！

三、 拓展——nth_element()函数讲解：

1、作用

将数组指定范围内第 k (从1开始编此号码) 小的元素（会更改原数组中所有元素的位置。如需不变，请预先备份）存于a[k-1]位置上，其左侧的元素都<=a[k-1]，右侧的元素都>a[k-1]，但左右两侧区间内的数据仍然凌乱。

2、用法

nth_element(起始地址，k-1，最后一个元素地址+1，自定义排序)；

举例：查找数组中第5小的元素

查找第 k 大的元素也是可以的。将第 k 大的问题转换为求第 n - k + 1 小的问题就可以了
例如：一个数组有10个元素，求第2大的数，那么它等同于求第10-2+1=9小的元素
